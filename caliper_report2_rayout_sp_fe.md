# 話すセリフ案

今回構築した仕組みは、

単一プロジェクト向けの自動化ではなく、

「再利用可能なワークフロー」

として整理できています。

今後は、

- テスト自動化手順
- スクリプト部品利用方法
- テスト観点テンプレート

などを Agent Skills 化し、

他メンバーや他プロジェクトでも、

専門知識なしで再利用できる形へ展開したいと考えています。

Agent Skills は、

生成AIへ作業手順やノウハウを定義し、

「誰でも同じ品質で作業できるようにする仕組み」

です。

これにより、

- ノウハウ共有
- 属人化低減
- 横展開
- 生産性向上

をさらに推進していきたいと考えています。
